# Design Guidelines: Arun Kumar Landscapes

## Design Philosophy
Premium architectural aesthetic inspired by Apple, Aesop, and Architectural Digest. The design prioritizes clean whitespace, elegant spacing, smooth transitions, and a calm, luxurious professional feel.

## Color Palette
- **Primary Background**: #F7F7F7 (soft grey)
- **White**: #FFFFFF
- **Text/Dark**: #222222 (charcoal)
- **Accent**: #9BAEA5 (muted sage green)
- **Overlays**: 20% dark overlay on hero images

## Typography
- **Headings**: Montserrat or Poppins
- **Body Text**: Inter or Lora
- Hierarchy emphasizes visual balance and elegant spacing

## Layout & Spacing
- Mobile-first responsive design
- Generous whitespace with architectural rhythm
- Spacing prioritizes breathing room and visual calm
- Use Tailwind units: primarily 4, 8, 12, 16, 20, 24 for consistency

## Global Components

### Navigation
- Sticky navbar with smooth scroll behavior
- Clean, minimal design
- Responsive mobile menu

### Footer
- 4-column elegant layout
- Links, contact info, social media, newsletter

## Animations
- Subtle fade-in effects using AOS library or custom CSS
- No heavy or distracting animations
- Smooth transitions on hover states (cards lift, images zoom)
- Scroll animations should feel natural and calming

## Images
- **Hero Images**: Full-width architectural/landscape photography with professional quality
- **Project Cards**: High-quality thumbnails of completed work
- **About Page**: Founder portrait, team photos
- **Gallery Images**: Professional architectural photography showcasing projects
- All images use placeholder URLs (Unsplash) for client replacement

## Component Behavior
- Cards: Subtle shadow, hover lift effect (4-8px translate)
- Buttons: Sage green accent, smooth hover transitions, blurred backgrounds when over images
- Forms: Clean borders, focused states with sage accent
- Modals: Elegant lightbox for image galleries

## Quality Standards
Spacing, typography, rhythm, and visual balance are top priorities. Every element should feel premium, architectural, calming, and client-ready.
